# Аналіз бізнесів



Аналіз існуючих бізнесів проподиться для того, щоб краще ознайомитись зі сферою, яку учасники обрали в якості пріорітетної для створення власного бізнесу.  
Якщо привести аналогію - діти спочатку все розбирають, а лише потім вчать будувати.  
Аналіз - це і є "розібрати на частини". Тоді як синтез - "зібрати з частин ціле".  
  
Розбирати бізнеси ми будемо на такі самі елементи, які ми будемо створювати для "збірки" власного бінесу.  Для цього виділемо 5 великих частин:
* **напрямок (сфер діяльності)** - ми її обираємо для власного бізнеу як об'єднання наших сильних якостей та існуючих можливостей. Придивившить до існуючих бізнесів ми зможемо припутити, якими можливостями скористалися відповідні бізнеси і на які сильні сторони спиралися. В тому числі для того, щоб використати ці спостереження для своїх висновків.
* **мета діяльності в регіоні** - це не завжди буде "отримання максимального прибутку", бізнеси можуть ставити мету максимального розширення, тестування ніші, видавлення конкурентів, формування іміджу, мінімізація навантаження. Мету, яку перед собой певний бізнес ми можемо припутити, спостерігаючи за ним. При цьому подивишись які патерни поведінки відповідають яким типам цілей ми зможемо запозичити цей досвід для визначення цілей для свого бізнесу.
* **яку ціннність і для кого створює** - основа ведення бізнесу це створення цінності. Важливо оцінити для яких категорій цільової аудиторії працює бізнес та яку саме цінність для них створює.
* **які ресури витрачає для створення цінності** - цінність не створюється сама по собі. Для того, щоб продати товар його треба десь виготовити або придбати, привезти, розмістити, тощо. Розуміння того, які витрати несе бізнес в галузі, яку ви обрали, допомагає оцінити те, які ресурси вам потрібно буде нести.
* **які параметри треба контролювати керуючому бізнеса** - навіть якщо все добре описано та зрозуміло, потрібно контролювати поточний процес, адже "все що може піти не так - піде не так". Для цього важливо виділити ключові параметри, контроль яких дозволяє забезпечити стале функціонування бізнесу.  

## Якісна та кількісна оцінка  
Важливо зазначити, що одночасно робити загальну оцінку і детальні обрахунки практично не реально, в тому числі тому, що ці два підхода мають різні цілі.  Загальний, якісний огляд дозволяє зрозуміти та уявити загальні тенденції, структуру (елементи та їх зв'язки) бізнесу. Якісне розуміння ситуації дозволяє "не загубитися".  
В той час як кількісний аналіз дозволяє оцінити будь-який елемент детально і прийняти по ньому рішення. Проте це рішення буде певним чином впливати на інші елементи бізнесу і вже для цього потрібне якісне розуміння ситуації.
