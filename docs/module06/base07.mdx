# Податки
Структура податкової системи. Схема державної реєстрації. Робота із державними сервісами та реєстрами. Кабінет платника податків. Цифрові можливості B2G.
Українська податкова система ділиться на загальну та спрощену. На спрощеній системі оподаткування суб’єкт підприємницької діяльності сплачує лише єдиний податок Єдиний соціальний внесок. Дана система спеціально створеня для того, щоб малий бізнес не витрачав значні ресурси на забезпечення діяльності. Адже для невеликого бізнесу витрати на спеціальний персонал із ведення обліку можуть бути вище ніж витрати на основну діяльність.
Для спрощеної системи оподаткування існує 4 групи: 1 група - найбільш прості види діяльності; 2 група - більшість видів діяльності із надання послуг та продажу тварів для фізичних осіб; 3 група - для продажу товарів та надання послуг юридичним особам; 4 група - для оподаткування суб’єктів підприємництва в агрогалузі, які ведуть діяльність за місцем реєстрації без найманих працівників.
 	Єдиний соціальний внесок - обов'язковою ставкою є 22% від мінімальної заробітної плати.
