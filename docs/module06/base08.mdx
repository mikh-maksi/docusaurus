# ERP
  
**Робота із ERP (програма управління ресурсами). Практична робота із відповідними ERP з реалізації сценаріїв, які описані в попередніх заняттях: ігрове опрацювання по ролях.**
  

	Діяльність суб’єкта підприємництва характеризується постійним збільшенням складності і кількості процесів. В результаті вже достатньо через короткий період часу кількість процесів за якими потрібно вести облік стає такими, що їх складно охопити увагою однієї людини.
	Найпростішим засобом ведення обліку є використання для цього електронних таблиць. При цьому в сучасному технологічному рівні розповсюдження сервісів найзручнішим сервісом електронних таблиць є Google.Stylesheet.
	Проте електронні таблиці не дозволяють робити перехресних запитів даних. Саме для розв’язання цієї складності використовують спеціальний тип програмного забезпечення ERP.
	Використання ERP. сильно допомагає підприємцю, адже в них закладаються певні структури даних і схеми їх взаємодії. 
	Існує поведінкова особливості ділової активност, яка полягає в фокусуванні уваги, в першу чергу, на активні події. Тоді, як облік і поточний стан справ, сам по собі не створює запити і, через це, вкрай уважно, тримати увагу на повноті обліку. Ведення ERP дозволяє не допускати помилок, пов’язаних із відсутності уваги (забуття) важливих питань: клієнти, кадри, контрагенти, тощо.
