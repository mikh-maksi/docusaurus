# 6.1. Фінанси та облік

## Ліквідність
Важливість ліквідності. Управління грошовим потоком. Фінансові коефіцієнти. Особливість роботи із фінансовими ресурсами. Засоби роботи та особливості прийняття та реалізації рішень в галузі фінансів. Джерела фінансів (внутрішні та зовнішні інвестиції, запозичення.). Технології автоматизації фінансових операцій
 
## Облік
Фінансовий та бухгалтерський облік. Засоби для ведення бухгалтерського обліку. Фінансове планування та звітування. Робота із банківськими рахунками. Юніт-економіка по всіх юнітах. Основні фінансові документи: Balance, P&L, Cash Flow statements. Особлиаості роботи із РРО.
 
           	Облік – це процес фіксації показників реального процесу на матеріальних носіях.
Важливість обліку полягає в тому, що отримання інформації безпосередньо з реального процесу потребує набагато більше ресурсів, ніж отримання доступу до інформації на спеціальному матеріальному носії. Через це достатньо 1 раз отримати дані з реального процесу, а всі інші операції доступу вже здійснювати безпосередньо до отриманих даних.
Розрізняють різні види обліку відповідно до мети ведення обліку. Мета ведення обліку формує вимоги до інформації, яка збирається: для бухгалтерського обліку потрібна інформація для рознесення по бухгалтерським рахункам; для фінансового обліку збирається інформація для таких документів, як Балансовий звіт (Balance Statement), Звіт з грошового потоку (Cash&Flow Statement) та звіт щодо доходів та розходів (Profit&Losses Statement); для управлінського обліку потрібні дані, на основі яких будуть прийматися управлінські рішення; для податкового обліку потрібні дані, на основі яких будуть нараховуватися податки.
РРО – реєстратор розрахункових операцій та ПРРО – програмний реєстратор розрахункових операцій. Це елементи податкового обліку. Їх важливість полягає в тому, що фіксуються всі операції, які проводяться з готівкою та оплатою електронною картою. Без використання такого реєстратора достатньо просто отримувати гроші за товари та послуги і не відображати їх в даних для податкового обліку.
РРО це вимога податкової служби з метою забезпечення фіксації всіх операції для податкового обліку.
