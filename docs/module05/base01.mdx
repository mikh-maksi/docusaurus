# Бізнес-фреймворки  

Формування ядра бізнес-плану із використанням бізнес-фреймворків: SWOT-аналіз, БізнесДНК, Value Proposition Canvas, Unit Economics, DashBoard - складові деталізації рішень. Методи прийняття оперативних рішень.
Основи управління проєктами: Agile-маніфест. Використання методологій SCRUM, KANBAN для планування проєкту та їх реалізації. Статут проєкту. Робота із Бек-логом

	Бізнес-фреймворки - це готові набори мог до рішень, що приймаються, виконання яких дозволяє отримати працююче рішення. Використання бізнес-фреймворків значно спрощує роботу з підготовки та реалізації рішень. 
	Бізнес-фреймворки є прикладним засобом, тобто зазвичай бізнес-фреймворк спрямований на отримання результату у певній вузькій галузі бізнес-процесів. Через це не є розповсюджені узагальнюючі фреймворки.
	Саме для розв’язання такої проблеми (як перевірити повноту наявних рішень) розроблено бізнес-фреймвор АСПіРІн (аналіз, стратегія, продукт, ресурси, індикація), який дозволяє, завдяки мнемонічній абревіатури запам’ятати основні елементи організаційного процесу. При цьому, кожний з даних елементів вказує на певний базовий бізнес-фреймворк у відповідному елементі організаційного процесу.
	Аналіз - SWOT-аналіз, Стратегія - БізнесДНК, продукт - Value Proposition Canvas, Ресурси - Юніт-економіка, Індикація - Дашборд.
	Також, для забезпечення зниження когнітивного навантаження при підготовки інформації для фреймворків для кожного елементу розроблено ключові фрази: аналіз - який найперспективніший напрямок діяльності; стратегія - який результат має бути досягнення в найближчий період планування; продукт - яку цінність і для кого пропонуємо; ресурси - які ресурси потрібні для надання цінності зацікавлених; індикація - які показники необхідно контролювати для досягнення успіху.
