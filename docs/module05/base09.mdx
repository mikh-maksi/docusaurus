# Планування рекламних заходів

```
Плани - ніщо. Планування - все!
```
Реклама є сферою із великою невизначеністю, а також із високою вартістю дій в ній.  
Саме тому, в рекламі ті елементи, які можуть робитися без планування (операційне управління поточними справами) в рекламі без планування приводять лише до збитків і розчарувань.  
Тому, перед тим, як запускати рекламні кампанії їх потрібно спланвувати.  
План включає в себе визначення зв'язків між елементами: певні дії прзводяться до певних результатів. В результаті, якщо план спрацював - це означає, що зроблені припущення та побудови були правильними. А якщо не спрацював - треба їх переробити. При цьому в результаті можна побачити що саме з загального розуміння було помилковим.
