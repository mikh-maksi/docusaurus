# Артефакти

* Pitch Deck: Презентація з 5–10 слайдів, що базується на BMC та фінансовому плані.
* Текст 3-хвилинного пітчу: Опрацьований сценарій з яскравим "хуком" (гачком).
* Q&A-лист: База з 8–10 питань від потенційних партнерів/інвесторів та аргументованих відповідей.
* Відео-пітч: Запис виступу (аудіо або відео) для саморефлексії та зворотного зв'язку.
