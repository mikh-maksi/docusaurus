# 3.2. Моделі маркетингу та продажів

*Воронка продажів. Шлях клієнту (Customer journey map). Методи впливу на прийняття рішень. Скрипти продажів. *

## Маркетинг
	Важливою складовою бізнес-моделі є те, як здійснюється продаж. Як на першому побаченні не пропонують створити подружжя (в тому числі через те, що ймовірність відмови буде завеликою), так і в продажах не завжди пропонують відповідний товар, проте поступово підводять до цього.
	Для того, щоб забезпечити процес продажів важливо побудувати послідовність взаємодій між продавцем та покупцем та підготувати відповідні дії всіма необхідними для цього матеріалами (від логічного обгрунтування до потрібних промоматеріалів).

Воронка продажів дозволяє будувати процес продажів "згори вниз", тобто будувати з точки зору продавця.  
  
Бізнес-фреймворк, який називається мапа подорожі клієнта (customer journey map) дозволяє збирати враження кожного учаснику процесу.


<img src={module03_03} alt="AIDA" />

	Обидва бізнес-фреймворки (воронка продажів та мапа шляху клієнту) дозволяють описати точки взаємодії із клієнтом. Виділення окремих точок взаємодії із клієнтом дозволяє промоделювати можливий вплив на нього: яким чином створити умови так, щоб ймовірність прийняття рішення щодо покупки була найбільшою.
	Серед засобів впливу на клієнта можна виділити ті, які роблять інформацію щодо продукту більш зрозумілою (зменшують рівень когнітивного навантаження, необхідний для прийняття потрібного рішення), а також ті, які активують певні емоції для сприянню прийняттю рішення.
	Екологічним підходом щодо впливу на потенційних клієнтів є ті, які дають клієнту користь. Тобто якісними рекламними повідомленнями ви дозволяєте майбутньому клієнту збільшити обсяг ресурсів, який він має в управлінні.
	Для підготовки рекламних матеріалів важливо визначити цінність, яку товари приносить тому, хто його купує; цінність, яку потенційний клієнт втрачає, якщо не купує (або загрозу, яка залишається діяти). А далі - сформувати опис того, як цю цінність отримає покупець при споживанні продукту.
	Одним з розповсюджених засобів, що підтвердив власну ефективність для забезпечення ефективних продажів є скрипт продажів. Скрипт (сценарцій) продажів - це опис того, як здійснюється продаж. В цілому можна порівняти скрипт продажів із сценарієм театральної вистави, адже продаж - це певне дійство, яке розігрується командою продавця з метою формування певного враження у потенційного покупця.
	При складанні скрипту продажів його автор має на принциповому рівні розв’язати питання виявлення, розвитку та підтримки інтересу клієнта, а також сформвати всю необхідну інформацію яка потрібна менеджеру з продажів для спілкування із клієнтом.
	Скрипт продажів не є догмою, проте чітке дотримання скрипту продажів має давати певний рівень конверсій в продаж і, якщо менеджер бере на себе відповідальність відходити від записаного скрипту, то його показники мають бути кращими за стандартні.
