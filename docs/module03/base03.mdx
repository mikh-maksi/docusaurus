# UI/UX. Когнітивне навантаження.

	Одним з напрямків роботи психології є вивчення того, як працює свідомість людини. І одним з важливих параметрів, який описує те, як людина сприймає інформацію є рівень когнітивного навантаження. Автором цього напрямку в психології є Джон Свеллер. Він виділив ключові принципи того, як працює когнітивне навантаження 
	В прикладному аспекті теорію когнітивного навантаження застосовують для побудови ефективних інтерфейсів. В галузі створення цифрових продуктів сферою застосування є UX-дизайн (UX - від User eXperience, тобто користувацький досвід). Те, що користуватися існуючими сайтами зручно ми завдячуємо саме даній дисципліні.
	Для роботи із клієнтом ми виділимо те, як порівняти ситуації з точки зору рівня когнітивного навантаження:
Вибір з декількох варіантів потребує меншого рівня когнітивного навантаження, ніж формування власного варіанта.
Декілька кроків, оцінювана складність яких зрозуміла потребують меншого рівня когнітивного навантаження ніж 1 крок, проте складність якого незрозуміла.
