# Вступ

**Базовий цифровий зошит** - це методичний прийом для роботи із підготовки бізнес-плану та, в подальшому, роботи зі створеним суб'єктом підприємництва.  
Одну з найважливіших задач, яку він виконує - це зібрання уваги щодо ведення бізнесу в одну точку.  
В базовому цифровому зохиті зібрано все, що потрібно для ведення бізнесу:  
1. Аналіз ситуації.  
2. Ключові ціннісні питання.  
3. Опис продукту.  
4. Фінансові та інші ресурсні розрахунки.  
5. Схема контролю поточних показників.   

Важливо зазначити, що у людини є обмеження на роботу із інформацією - людина не може робити все й одразу, а також для людини важливою є зв'язність і поступова деталізація.  
Саме тому ми будемо використовувати рівні деталізації для кожного елементу:
1. Дізнатися.  
2. Продумати.  
3. Записати.  
4. Структурувати.  
5. Структурувати за вимогами. Візуалізувати.  
6. Автоматизувати.  

В результаті навчання ви будете мати уявлення про всі елементи які потрібні для ведення підприємницької діяльності, а також знати рівень деталізації кожного елемента конкретно вами.
