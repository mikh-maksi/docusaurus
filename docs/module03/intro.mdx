# Вступ

Зв'язати потреби клієнта, описати та оргіназувати процеси, які дозволяють виробити та доставити, прорекламувати та продати продукт ще не є достатніми, щоб даний процес можна було назвати бізнесом.  
Можливо, це якийсь суспільно-значущий процес, благодійність, громадська діяльність або робота органу державного упраління.  
  
Для того, щоб такий процес став бізнесом він має давати прибуток.  Тобто ресурсів, після протікання такого процесу, сумарно має бути більше ніж до нього.  
  
Для того, щоб забезпечити прибуток необхідно врахувати всі елементи, а також порахувати всі витрати та всі очікувані доходи.
